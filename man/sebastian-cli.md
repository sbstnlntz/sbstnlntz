<h2>USAGE</h2>
<p>
&nbsp;&nbsp;&nbsp;&nbsp;sebastian [options]
</p>

<h2>OPTIONS</h2>
<p>
<pre>
--code         Activate development mode
--backend      Backend-focused logic
--frontend     UI and component focus
--design       UX, flow and interface thinking
--review       Code review mode
--build        Deployment, Docker, environments
--debug        Troubleshooting mode
--coffee       Boost energy + creativity
--learn        Explore new technologies
--philosophy   Open sebastian(7)
</pre>
</p>

<h2>EXAMPLES</h2>
<pre>
sebastian --code
sebastian --code --backend
sebastian --build --debug
sebastian --philosophy
</pre>
